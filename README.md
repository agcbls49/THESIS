# THESIS
Voice-Guided Indoor Object Detection Android Application for Visually Impaired Users

## Dataset
A custom dataset was used for the project and the images are taken under different lighting conditions. 

**[The dataset source can be found here](https://drive.google.com/drive/folders/1OVjp1W8MENmfdceN69IOThPcHpZhV21c?usp=sharing)**

> [!NOTE]
> **The data listed on the table below is subject to change and was updated in April 27, 2026.
> Annotation started in April 23, 2026. The class list below are also subject to change with
> 10 more classes to be added as synthetic data.**
